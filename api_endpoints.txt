## Notes

- All routes under `/api/` (except public auth and market endpoints) require authentication
- Admin routes require additional administrative privileges
- WebSocket endpoints support real-time market data updates
- The API uses JWT tokens and API keys for authentication
- All routes are protected with security headers, CORS, and request logging middleware
